# Animate Any Line Plot

A lightweight Python tool to create animated GIFs that visualize the
**temporal evolution of a time series** by progressively revealing data
from left to right along the x-axis.

This is useful for:
- Scientific time series visualization
- Data storytelling and reports
- Teaching and presentations
- Social media and documentation animations

## Features

- Accepts **y-only** or **(x, y)** time series
- Smooth left-to-right data reveal
- Optional sliding time cursor
- Fixed axes (no jitter)
- High-quality looping GIF output
- Spyder, Jupyter, and terminal compatible

---

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/your-username/timeseries2gif.git
cd timeseries2gif
pip install -r requirements.txt
```
